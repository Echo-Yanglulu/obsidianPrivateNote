# 概述
一种轻量级[[标记语言]]，
使用易读易写的纯文本格式编写文档
# 语法
## 标题
jakls
> er
>> jiow

- wef

## 段落

直接使用两个空行将一段文本包裹

## 行内代码

## 强调
粗体：关键字
斜体：观点、过程
## 列表
一个-
无序列表：无重要性等先后顺序的内容
有序列表：有顺序（重要性、创建时间）的内容

## 分割线
三个-
用于：分隔原文与注释
## 链接
带说明：方括号中是说明，圆括号中是链接
不带说明：尖括号
