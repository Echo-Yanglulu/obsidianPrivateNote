# 概述

# 对象
定义
	1. 一种复合值（汇聚多个值），允许按名字读写这些汇聚的值
	2. 任何不是[[String]],[[Number]], Boolean, null, undefined, [[Symbol]], [[BigInt]]的值都是对象。前三者不是对象，也类似不可修改的对象
	3. 属性的无序集合
特性
	1. 动态：可动态增删属性。
