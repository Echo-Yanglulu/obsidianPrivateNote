# 概述
定义
	1. 使用<u>整数</u>作为元素索引的[[类列表对象]]。
		1. 使用<u>字符串</u>作为索引的叫[[关联数组]]。
		2. 通过方括号或点语法，读写数组的<u>非整数</u>索引元素时，读写的实际上是数组对象的**属性集合**[^1]上的变量。
原型中提供了操作方法
特性
	1. 动态：数组的长度，元素的[[数据类型]]不固定
	2. 数据在内存中可以不连续：JS数组不一定是密集型的。
如果以上特性不适合使用场景，可使用[[TypedArray]]
