# 小结
1. 浏览器自身展示文档时
	1. 有**BOM**与**DOM**；
	2. 动画，canvas，svg是文档的普通内容；
	3. **表单**是文档的特殊内容涉及与用户信息的收集；
	4. **事件**是用户交互的反馈
	5. **网络请求**是连接其他电脑的方式
	6. 有些应用程序通过网络请求前需要进行客户端**检测**，然后请求适配的数据。有时可能需要将得到的资源需要**存储**在客户端。
	7. 应用本身丰富带来的代码量较大，**命名冲突**等问题，由模块化作为解决方案。带来的**任务较多**，可由工作者线程解决。
