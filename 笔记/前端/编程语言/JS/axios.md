# 概述
定义：一个基于 [[promise]] 的[[网络请求库]]，作用于node.js 和浏览器中。
	1. 在服务端使用原生[[node|node.js]]的http模块
	2. 在客户端则使用[[网络请求#XHR对象|XMLHttpRequests]] 
特性
	1. isomorphic。即同一套代码可以运行在浏览器和node.js中
