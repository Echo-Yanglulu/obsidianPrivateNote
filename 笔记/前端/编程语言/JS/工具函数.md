防抖
在规定时间后执行函数，如果规定时间内触发，则重置定时
```js
  const debounce = (fn, second) => {
    let timeid = false;
    return (...arg) => {
      if (timeid) {
        clearTimeout(timeid);
        timeid = false;
      }
      timeid = setTimeout(fn, second);
    };
  };
```
节流
在规定时间内只执行一次【可以在开始，也可以在结束】
```js
  const throttle = (fn, delay) => {
    let timeid = null;
    return (...args) => {
      if (!timeid) {
        // 如果希望在规定时间开始时执行，fn放在外面；
        // 希望在规定时间结束时执行，fn放在定时器里面
        fn.call(this, ...args);
        timeid = setTimeout(() => {
          clearTimeout(timeid);
          timeid = false;
        }, delay);
      }
    };
  };
```
