# 真机远程代码调试/真机调试
1. 真机（手机）
	1. IOS, android
2. 开发主机
	1. macOS, windows
3. 调试工具
	1. Safari Developers Tools, chrome Devtools 
4. USB数据线
	1. 连接真机与开发主机

IOS调试
	1. 打开手机safari的“网页检查器”
	2. 打开开发主机safari中的preferences->advanced->show develop menu in menu bar.
	3. 数据线连接真机与开发主机
	4. 打开开发主机中菜单栏的develop
Android 调试
	1. 手机开发者选项->开启USB调试
	2. 开发主机地址栏输入chrome://inspect，确保浏览器开启了“发现USB设备”
	3. 选择目标页面下的inspect。
