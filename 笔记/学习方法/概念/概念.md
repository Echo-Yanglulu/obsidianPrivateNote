# 概述
定义
	1. 反映[[客观事实]]的[[本质属性]]/特有[[属性]]的[[思维形式]][^1]。
	2. 与某个抽象事物相关的思想、原则
# 起源
从现象抽象的产物。人类认知的思维体系中最基本的单位。
人通过对[[客观事实]]进行[[比较]]，[[分析]]，[[综合]]，[[抽象]]，[[概括]]等方法[^2]，得到对客观事实的本质属性的认识，形成对应的科学概念。如果只是知道名称，并不等于对事物有了科学[[概念]]。【就是说，没有进行那些方法，就没有对本质属性的认识？没有形成对应的科学概念？】
