# 概述
沟通不是表达能力的问题，而是心理建设的问题。
# [[原则]] 
1. 开放性：学会说“我们”，团结任何人
2. 目标：更有目标的人，能掌握主动权
3. 建设性：从“我要”到“我来”

# [[策略]] 
1. 先听再说
2. 听话听音：听懂别人没说的想法
3. 积极回应：让不友好的人好好说话
