# 过程及应用
1. 发现客观事实的**本质属性**
2. 对事物的**联系、性质**做出判断，总结为概念
3. 构成**命题**
4. 由命题之间的**联系**推理出新知
5. 再对客观事实做**全面整体的思考**，最终系统认知客观事实
